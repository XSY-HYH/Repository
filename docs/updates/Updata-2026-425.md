# Changelog - April 25, 2026

## New Features

### 1. Rate Limiting Protection
- Added rate limiting protection service (`RateLimitProtectionService`)
- Different from DDoS protection (which blocks IPs)
- When request frequency exceeds threshold:
  - Temporarily pauses service
  - Stops accepting new requests
  - Automatically resumes after configured duration
- Configuration options:
  - `RateLimitProtection` - Enable/disable rate limiting
  - `RateLimitRequestsPerSecond` - Max requests per second threshold
  - `RateLimitPauseMinutes` - Pause duration in minutes

### 2. Admin Server Settings Panel
- Added server settings page in admin panel
- Quick configuration without directly editing config file
- Categorized settings:
  - DDoS Protection settings
  - Rate Limiting settings
  - File Upload settings
  - File Preview & Download settings
  - Network settings (HTTP/HTTPS)
  - System settings (Notifications, Auto-restart)
- Real-time rate limit status display
- Force resume button when service is paused

### 3. SHA256 Hash Display
- Files and directories now display SHA256 hash
- Hash shown in faded style next to item name
- Click to copy hash to clipboard
- Helps verify file integrity

### 4. Access Count Optimization
- Access counts no longer written in real-time
- Counts accumulated in memory
- Written to disk only on server exit
- Improves performance and reduces disk I/O

### 5. Auto-Restart Configuration
- Added auto-restart configuration options
- `AutoRestart` - Enable/disable auto-restart on crash
- `MaxRestartAttempts` - Maximum restart attempts
- `RestartCount` - Current restart count

## Bug Fixes

### 1. FileSystemWatcher Disposed Object Error
- Fixed "Cannot access a disposed object" error
- Error occurred during application shutdown
- Added null checks and disposed state verification
- Added try-catch blocks around FileSystemWatcher operations
- Ensures graceful shutdown without exceptions

### 2. Missing Configuration Options in Admin Panel
- Added missing configuration options to server settings:
  - `AllowOverwrite` - Allow file overwrite
  - `AllowRootUpload` - Allow upload to root directory
  - `PreviewEnabled` - Enable file preview
  - `MaxDownloadSizeMB` - Max download size
  - `HttpEnabled` - Enable HTTP
  - `HttpsEnabled` - Enable HTTPS
  - `HttpsRedirectEnabled` - Enable HTTPS redirect
  - `Notification` - Enable Windows notifications
  - `MaxRestartAttempts` - Max restart attempts
