# Update Log - April 26, 2026

## New Features

### 1. PROXY Protocol Support
- Added PROXY Protocol parsing service (`ProxyProtocolService`)
- Supports PROXY Protocol v1 and v2 formats
- Retrieves real client IP through reverse proxies like frp
- Configuration options:
  - `ProxyProtocolEnabled` - Enable/disable PROXY Protocol support
- Integrated with rate limiting, circuit breaker, and request header filtering services
- Debug mode displays raw packet data and PROXY header parsing details

### 2. Debug Mode
- Added `--debug` startup parameter
- Displays DEBUG level logs when enabled
- Log output includes:
  - Raw received packet data (hexadecimal format)
  - PROXY Protocol header parsing details
  - Connection ID for request tracking
- Log color: Cyan

### 3. YAML Configuration File
- Migrated configuration format from JSON to YAML
- Supports dynamic comment replacement (based on Language setting)
- Default English comments, automatically replaced with Chinese comments when language is set to Chinese
- Configuration file: `Config.yml`
- Backward compatible: automatic migration from old configuration

## Feature Optimizations

### 1. Config Manager Refactoring
- `ConfigManager` supports YAML serialization/deserialization
- Removed duplicate ConfigManager instances
- Fixed configuration change throttling logic
- Fixed file watcher duplicate trigger issues

### 2. Logging System Optimization
- Added `LogDebug` method
- Supports debug mode toggle
- Fixed duplicate log display issues
- Unified log output format

### 3. PROXY Protocol Middleware
- Processes PROXY headers at Kestrel transport layer
- Correctly strips PROXY headers before passing to TLS/HTTP handlers
- Fixed TLS handshake issues on HTTPS port
- Fixed middleware registration order (PROXY processing before TLS decryption)

## Bug Fixes

### 1. PROXY Protocol and HTTPS Compatibility Issue
- Fixed TLS handshake failure caused by incorrect middleware registration order

### 2. Configuration Comment Replacement Issue
- Fixed naming convention mismatch during YAML deserialization

### 3. Port Binding Conflict
- Fixed duplicate port binding when PROXY Protocol is enabled
