# Changelog - April 5, 2026

## New Features

### 1. Disk Usage Limit
- Added `MaxDiskUsagePercent` configuration option
- Limits the disk usage percentage for the repository
- Checks estimated usage before upload, rejects if limit exceeded
- Returns HTTP 507 status code indicating insufficient disk space

### 2. Protection Feature Toggle
- Added `ProtectEnabled` configuration option
- Can disable protection feature to speed up startup
- When disabled, protected directories are accessible without verification
- "Invisible" feature is not affected

### 3. Root Directory Upload Control
- Added `AllowRootUpload` configuration option
- Upload to repository root directory is disabled by default
- Can be enabled as needed
