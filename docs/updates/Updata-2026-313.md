# Changelog - March 13, 2026

## Update 1: Automatic SSL Certificate Generation

### New Features
- When HTTPS is enabled but no certificate path is configured, the program automatically generates a self-signed certificate
- Added `Domain` configuration option to specify certificate domain
  - Domain set: Certificate issued to specified domain
  - Domain not set: Certificate issued to 127.0.0.1
- Certificate validity: 5 years
- Includes SAN extension, supports wildcard domains

### Configuration Updates
```json
{
  "Domain": ""
}
```

### File Changes
- `Services/CertificateGenerator.cs` - SSL certificate generation service
- `Models/Config.cs` - Added Domain configuration
- `Program.cs` - Integrated certificate generation logic
- `Config.json` - Added Domain configuration option
- `help.txt` - Updated configuration documentation

## Update 2: Certificate Format Changed to CRT

### Changes
- Certificate format changed from PFX to CRT/KEY (PEM format)
- Certificate file naming: `webdata_{domain}.crt`
- Private key file naming: `webdata_{domain}.key`

### File Format
- **Certificate file (.crt)**: PEM format X.509 certificate
  ```
  -----BEGIN CERTIFICATE-----
  ...
  -----END CERTIFICATE-----
  ```

- **Private key file (.key)**: PEM format RSA private key
  ```
  -----BEGIN RSA PRIVATE KEY-----
  ...
  -----END RSA PRIVATE KEY-----
  ```

### Advantages
- CRT/KEY format is more universal, compatible with Nginx, Apache, and other mainstream servers
- PEM format is text file, easy to view and transfer
- No password protection needed, simplified configuration

### Upgrade Notes
1. Delete old `.pfx` certificate files
2. Program will automatically generate new `.crt` and `.key` files
3. `HttpsCertificatePath` in configuration will be automatically updated to `.crt` file path
4. `HttpsCertificatePassword` configuration is no longer needed (PEM format requires no password)

### File Changes
- `Services/CertificateGenerator.cs` - Changed to generate PEM format certificates
- `Program.cs` - Updated certificate path handling logic
