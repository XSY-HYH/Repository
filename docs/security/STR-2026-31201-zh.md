## 发现的漏洞

### 漏洞1: HRVU-202631205 - 未授权文件上传到根目录

| 属性 | 内容 |
|------|------|
| 漏洞编号 | HRVU-202631205 |
| 漏洞类型 | 未授权文件上传 |
| 风险等级 | 高危 |
| 受影响端点 | `POST /api/upload/`, `POST /api/upload` |
| 漏洞描述 | 攻击者可在无认证情况下上传文件到仓库根目录 |
| 影响范围 | 可能上传恶意文件、覆盖配置文件、污染目录结构 |

#### 修复方案
在 `UploadController.cs` 中添加根目录上传限制：

```csharp
if (string.IsNullOrEmpty(folderPath) || folderPath.Trim('/') == "")
{
    _logger.LogWarning($"尝试上传到根目录，客户端IP: {clientIP}");
    return BadRequest("不允许上传到根目录");
}
```
