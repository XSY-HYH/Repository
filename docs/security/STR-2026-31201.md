## Discovered Vulnerabilities

### Vulnerability 1: HRVU-202631205 - Unauthorized File Upload to Root Directory

| Attribute | Content |
|------|---------|
| Vulnerability ID | HRVU-202631205 |
| Vulnerability Type | Unauthorized file upload |
| Risk Level | High |
| Affected Endpoints | `POST /api/upload/`, `POST /api/upload` |
| Description | Attackers can upload files to repository root directory without authentication |
| Impact | May upload malicious files, overwrite configuration files, pollute directory structure |

#### Fix Solution
Add root directory upload restriction in `UploadController.cs`:

```csharp
if (string.IsNullOrEmpty(folderPath) || folderPath.Trim('/') == "")
{
    _logger.LogWarning($"Attempt to upload to root directory, client IP: {clientIP}");
    return BadRequest("Upload to root directory is not allowed");
}
```

#### Fix Status
✅ Fixed
